SVG ILLUSTRATION SYSTEM

When external image assets are unavailable, inline SVG can still feel premium.

- Build illustrations from simple geometry, strong silhouette, and restrained color count.
- Reuse stroke widths, corner radii, and gradient logic across illustrations.
- Prefer symbolic scenes over cluttered literal scenes.
- Optimize for hero readability first, detail second.
- If the page needs imagery now, ship clean SVG instead of waiting on unavailable image tooling.
