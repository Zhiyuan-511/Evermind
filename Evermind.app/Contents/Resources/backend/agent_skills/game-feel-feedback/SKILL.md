GAME FEEL FEEDBACK

Games fail even with working mechanics if player input feels dead.

- Every important input should cause visible feedback within one beat.
- Use juice: scale pops, hit flashes, particles, screen shake, score ticks, or HUD reactions.
- Keep control latency low and motion readable.
- Difficulty can be simple; feel cannot be flat.
- If time is limited, simplify scope but preserve responsiveness and feedback.
