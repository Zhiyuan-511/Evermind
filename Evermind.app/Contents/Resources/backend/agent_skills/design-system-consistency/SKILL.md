DESIGN SYSTEM CONSISTENCY

Quality drops fast when each section looks like a different product.

- Reuse the same spacing rhythm, corner radius family, and elevation logic.
- Primary, secondary, and destructive actions must be visually consistent across screens.
- Typography roles should be stable: page title, section title, body, caption, data value.
- If one section uses a different visual language, justify it with product function.
- In review, call out inconsistent controls even when each piece looks fine alone.
