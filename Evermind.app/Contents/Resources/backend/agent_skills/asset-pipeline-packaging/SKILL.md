ASSET PIPELINE PACKAGING

Treat assets as a production pipeline, not loose files.

- Name outputs by role and variant: `hero-main`, `enemy-flyer`, `slide-cover`, `logo-dark`.
- Keep a manifest mindset: source, intended use, size/aspect ratio, and final export target.
- Prefer consistent canvas sizes within one family of assets.
- When packaging sprites or icons, align padding and baseline rules before export.
- If tooling is missing, still output a clean asset manifest and folder plan instead of vague notes.
