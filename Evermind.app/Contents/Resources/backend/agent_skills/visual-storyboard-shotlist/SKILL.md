VISUAL STORYBOARD SHOTLIST

For image-heavy work, think in shots and coverage.

- Break the request into 3-6 shots or frames with a clear role for each.
- Define wide / medium / close-up coverage where useful.
- Include subject action, camera framing, mood, and key prop or background cues.
- For campaigns or decks, ensure the shotlist covers hero, feature detail, proof, and closing image.
- If one hero image is not enough, output a reusable shot plan instead of a single vague prompt.
