COMMERCIAL UI POLISH

Build for ship quality, not demo quality.

- Visual hierarchy must be obvious within 3 seconds.
- Every primary CTA must stand out from secondary actions.
- Use contrast, spacing rhythm, and section pacing to avoid "flat wall of content".
- Avoid dead zones: every major viewport segment should either inform, persuade, or advance the task.
- Ensure hover/focus/active states exist for all obvious controls.
- Treat copy, layout, and motion as one system: if one feels generic, the product feels generic.

Before finishing, sanity-check:
- hero is distinctive
- typography scale is intentional
- cards/sections align cleanly
- mobile collapse still feels designed, not just stacked
