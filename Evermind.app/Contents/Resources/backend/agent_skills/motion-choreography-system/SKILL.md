MOTION CHOREOGRAPHY SYSTEM

Animation should guide attention, not decorate emptiness.

- Define motion roles: page load, section reveal, hover, state change, success/failure.
- Use stagger and pacing to create rhythm across repeated elements.
- Big motion belongs to major transitions; micro-motion belongs to controls and feedback.
- Prefer 2-4 coherent motion patterns reused well over many unrelated effects.
- In review, reject motion that feels noisy, laggy, or disconnected from user intent.
