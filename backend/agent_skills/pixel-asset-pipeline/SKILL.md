PIXEL ASSET PIPELINE

Pixel assets need systemic constraints or they fall apart fast.

- Lock grid size, outline policy, shading depth, and palette before generating many assets.
- Separate gameplay readability from decorative detail.
- Characters, props, and tiles should share horizon, light direction, and scale logic.
- Plan sprites for animation states up front: idle, move, hit, defeat, UI icon.
- If generating prompt packs, specify palette count, resolution, and side/front views explicitly.
