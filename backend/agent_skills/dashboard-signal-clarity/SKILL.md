DASHBOARD SIGNAL CLARITY

Dashboards fail when density beats clarity.

- Lead with 3-4 high-signal KPIs before secondary detail.
- Important filters/actions must be visible without hunting.
- Tables and charts must answer different questions, not duplicate each other.
- Trend indicators must be scannable by color and direction.
- Empty states and inactive controls should still look intentional.

During review/testing:
- verify at least one interactive control changes visible state
- reject if charts, filters, or tabs look present but do nothing
