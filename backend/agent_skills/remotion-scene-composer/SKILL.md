REMOTION SCENE COMPOSER

Treat motion work like a composition, not a single prompt.

- Break the video into explicit scenes with duration, objective, visual hook, copy, and transition.
- Define a render plan: aspect ratio, fps, total duration, background system, caption style, and audio intent.
- Keep every scene reusable: one scene should have a clear input, output, and visual delta.
- Prefer strong opening frames, readable mid-sequence information density, and a decisive ending CTA.
- For deliverables, return:
  - a scene table
  - timing notes
  - caption / on-screen text
  - transition notes
  - render/export recommendations
